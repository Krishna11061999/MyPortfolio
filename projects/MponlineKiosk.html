MPOnline Kiosk

  • Project: MPOnline Kiosk Application
  • Technology: Developed using ASP.NET Web Forms with SQL Server as backend
  • Purpose: Facilitate G2C services for citizens of Madhya Pradesh via public kiosks
  • API Integrations:
    • Samagra API – to validate residents of Madhya Pradesh using Samagra ID
    • PAN API – to validate users across India using PAN card details
  • Key Features:
    • Real-time user verification through integrated APIs
    • Secure data submission and validation for government schemes
    • Designed for easy use in rural and urban kiosk centers
  • Impact:
    • Streamlined service delivery at citizen touchpoints
    • Improved accuracy and trust with identity validation
